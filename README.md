## Run the Program
clone this repository
```sh
   git clone https://Newton-Nganga/vizit.git
```
Install dependencies
```sh
   cd vizit && npm install
```
Run the application
```sh
   npm start
```
This should run the server @localhost:4000  

open the browser and type in the url localhost:4000
